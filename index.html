About Me

Born and raised in Hong Kong, I am someone who is familiar with many different languages and cultures. Having been involved with computers since my childhood, I was inspired to begin creating my own applications and to get involved with programming. I began with the very basics, creating simple web pages with services like google sites and then moving on to bigger things like game development and mobile application development. Now, I aim to give back to the community by creating the very things that helped inspire me in my youth. As an aspiring developer, my adaptability allows me to work with both AGILE and Waterfall development methodologies, and my problem solving skills and desire for self-improvement make me a dedicated and valuable member of any team.

More About Me

One of my earliest memories was playing the original Lego Creator game on a Kindergarden pc. Even then, I was entranced by how the game worked, with each of its component functions coming together to create a program that allowed the user to create infinite possiblities by combining many smaller individual parts. To me, Coding is an art form that is very much similar, allowing me to create individual parts that function individually but only really come to life when they are combined to form the final product. The simplicity of the form and function are aspects of my experiences which I take as a metaphor for programming, taking many simpler functions and components and combining them to create something greater than the individual parts.

While coding is the framework that holds everything together, the UI/UX and graphics of the application are what the consumer ultimately sees, and is an equally important part of development. The UI of a good application has to be both aesthetically pleasing and also intuitive, allowing users to understand operations and to manipulate the application at will. To avoid meaningless clutter and to create an attractive user interface, I prefer to use clean lines and color theory to accentuate a crisp and minimalist aesthetic as required for the purposes of each application.

Some of my past projects

Chron

Chron is a 2D platformer made using FXGL, a Java/JavaFX/Kotin Game Library Engine which uses a superset of Java. Built from the ground up for Windows/Mac/Linux/Android/IOS devices, Chron uses integrated Jbox2D system for physics, movement and collisions, allowing for both fluid and rigid-body simulations in real time.

Acolyte

Acolyte is a game made in the Unity game engine with C# over the period of 72 hours for the 43rd Ludum Dare game jam, under the theme of “Sacrifices must be made.” It is a 2D puzzle game with quests and minigames that are procedurally generated so that the player must investigate and discover a different ritual every time in order to bring about the apocalypse.

Turtle's Bizzare Adventure

Developed over the period of 72 hours for the 2019 Ukie Green Game Jam under the themes of climate change and sustainability, Turtle's Bizzare Adventure is a side scrolling horizontal runner developed with the unity engine using C#. The game uses Procedurally generated obstacles which help to ensure game replayability and a time based scoring system which encourages player retention through competition.
